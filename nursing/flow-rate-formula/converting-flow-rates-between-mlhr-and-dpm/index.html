Converting flow rates between mL/hr and dpm

How do you convert IV flow rates? In the previous video, we looked at converting between millilitres (ml) and drops. This time we are converting not only the volumes, (drops and mls) but the units for time as well.

Assuming you are using macrodrops 20 drops/ml, complete the following equations by filling in the gaps. Answers will be given at the end.

  1. $15 mL/hr = \fbox{?} ~ dpm$
  2. $60 mL/hr = \fbox{?} ~ dpm$
  3. $450 mL/hr = \fbox{?} ~ dpm$

Answers

  1. $15 mL/hr = 5 dpm$
  2. $60 mL/hr = 20 dpm$
  3. $450 mL/hr = 150 dpm$

Note that $240 ~ mL \times 20 = 4800 ~ drops$ and $2 ~ hours \times 60 = 120 ~ mins$. Fill in missing values for each empty box.

Question 1

  1. $120 ~ mL \times \fbox{?} ~ = \fbox{?} ~ drops$
  2. $1 ~ hour \times \fbox{?} ~ = \fbox{?} ~ mins$

Answer

  1. $120 ~ mL \times 20 = 2400 ~ drops$
  2. $1 ~ hour \times 60 = 60 ~ mins$

Question 2

  1. $240 ~ mL \over 2 ~ hours$ $\times$ $\fbox{?} ~ drops \over \fbox{?} ~ mins$ $=$ $4800 ~ drops \over 120 ~ mins$ = $\fbox{?} ~ drops/mins$

Answer

  1. $240 ~ mL \over 2 ~ hours$ $\times$ $\fbox{20} ~ drops \over \fbox{60} ~ mins$ $=$ $4800 ~ drops \over 120 ~ mins$ = $\fbox{4} ~ drops/mins$

Question 3

  1. $120 ~ mL \over 1 ~ hour$ $\times$ $\fbox{?} ~ drops \over \fbox{?} ~ mins$ $=$ $\fbox{?} ~ drops \over 60 ~ mins$ $=$ $\fbox{?} ~ drops/mins$

Answer

  1.  $120 ~ mL \over 1 ~ hour$ $\times$ $\fbox{20} ~ drops \over \fbox{60} ~ mins$ $=$ $\fbox{2400} ~ drops \over 60 ~ mins$ $=$ $\fbox{4} ~ drops/mins$

Given that the flow rate is 40 dpm (drops per minute).

  1. What is the flow rate in ml/minute?
  2. If the same flow rate continued for 60 minutes, how many drops would pass through?
  3. If the same flow rate continued for the same hour, how many mls would pass through?
  4. If 40 dpm, what is its equivalent in mL/hr?

Answers

  1. 2
  2. 2400 drops
  3. 120 mL
  4. 120 mL/hr

Questions

  1. Drops/ml is a flow rate
  2. There are 20 macrodrops in 1 ml
  3. There are 20 microdrops in 1 ml
  4. For macrodrops 30 ml/hr = 10 dpm
  5. For macrodrops 30 ml/hr = 600 drops/hr
  6. For microdrops 40 ml/hr = 120 dpm
  7. For microdops 20ml/hr = 1200 drops/hr
  8. For macrodrops 90dpm = 90ml/hr

Answers

  1. False
  2. True
  3. False
  4. True
  5. True
  6. False
  7. True
  8. False
