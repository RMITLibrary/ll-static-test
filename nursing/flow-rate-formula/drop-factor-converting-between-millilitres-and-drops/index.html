Drop factor: converting between millilitres and drops

Drops are just another unit for measuring the amount of fluid flowing into a patient’s system. Discover what ‘drop factor’ means, and how to convert between millilitres and drops.

There are 20 drops in 1mL, this is contained in one box. Imagine four of these boxes together.

  1. How many drops are in each mL?
  2. How many drops are there?
  3. How many mLs are there?
  4. What is the drop factor?

Answers:

  1. How many drops are in each mL? 20 drops
  2. How many drops are there? 80 drops
  3. How many mLs are there? 4 mLs
  4. What is the drop factor? 20 drops/mL

A tube drops 20 times into a beaker.
If you are using macrodrops 20drops/ml. How many mLs does these drops equal?

Answer: 0.5

Assuming you are using macrodrops 20drops/ml. Complete the following equations. There are two parts.

Part 1

  1. 1 mL = [ ] drops
  2. 40 mL = [ ] drops
  3. 100 ml = [ ] drops

Answers:

  1. 1 mL = 20 drops
  2. 40 mL = 800 drops
  3. 100 ml = 2000 drops

Part 2

  1. [ ] ml = 20 drops
  2. [ ] ml = 600 drops
  3. [ ] ml = 3500 drops

Answers:

  1. 1 ml = 20 drops
  2. 30 ml = 600 drops
  3. 175 ml = 3500 drops
