Page redirects

This page lists all the pages which have redirects to new locations.

  1. Organise: Make the parent page "Page redirects" under "Documentation" to keep all redirects in one place.
  2. Template and Slug: Select the "redirect" page template. A Redirect Path field will appear — fill this with the required path. You'll need to paste in a complete path, rather than just the slug to make the redirect function in the final static site. For example, rather than using:
    /essays/

    You should input:

    /assessments/essays/

    If the redirected page is moved, the new path will need to be manually re-entered into this field.

  3. Keywords: Add "Archive" to the keywords to prevent the page from being searched.

Note that none of the content is displayed, but it is retained for reference or potential reversal of changes. The redirect is currently set to 5 seconds.
